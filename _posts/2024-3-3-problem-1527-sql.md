## Query

```sql
SELECT
    *
FROM
    Patients
WHERE
    conditions LIKE 'DIAB1%' OR conditions LIKE '% DIAB1%'
```

## Editorial Solution

### Approach 1: Using Regular Expression Word Boundaries

```sql
SELECT patient_id, patient_name, conditions
FROM Patients
WHERE conditions REGEXP '\\bDIAB1.*';
```
