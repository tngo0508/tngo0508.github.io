## Problem Statement

[leetcode problem link](https://leetcode.com/problems/degree-of-an-array/description/?envType=company&envId=jpmorgan&favoriteSlug=jpmorgan-all)

## Solution [accepted]

```python
class Solution:
    def permute(self, nums: List[int]) -> List[List[int]]:
        res = []
        def backtrack(start, arr):
            if start == len(arr):
                res.append(arr[:])
                return
            for i in range(start, len(nums)):
                arr[i], arr[start] = arr[start], arr[i]
                backtrack(start + 1, arr) # common mistake I always make: start + 1 instead of i + 1
                arr[i], arr[start] = arr[start], arr[i]


        backtrack(0, nums)
        return res
```

- time: O(n\*n!)
- space: O(n\*n!)
