## Problem Statement

[leetcode problem link](https://leetcode.com/problems/smallest-number-with-all-set-bits/description/?envType=daily-question&envId=2025-10-29)

## Brute Force [TLE]

```python
class Solution:
    def smallestNumber(self, n: int) -> int:
        x = n
        res = 0
        while x > 0:
            res = (res << 1) | 1
            x = x >> 1

        return res
```

## Editorial

```python
class Solution:
    def smallestNumber(self, n: int) -> int:
        x = 1
        while x < n:
            x = x * 2 + 1
        return x
```
