## My solution

```python
class Solution:
    def pivotArray(self, nums: List[int], pivot: int) -> List[int]:
        N = len(nums)
        res = [0] * N
        idx = 0
        for num in nums:
            if num < pivot:
                res[idx] = num
                idx += 1
        for num in nums:
            if num == pivot:
                res[idx] = num
                idx += 1
        for num in nums:
            if num > pivot:
                res[idx] = num
                idx += 1
        return res
```
