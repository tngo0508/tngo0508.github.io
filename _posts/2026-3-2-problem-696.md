## Problem Statement

[leetcode problem link](https://leetcode.com/problems/count-binary-substrings/description/?envType=company&envId=jpmorgan&favoriteSlug=jpmorgan-all)

## Solution [Accepted]

Expand the window to the left and right of the current index.

```python
class Solution:
    def countBinarySubstrings(self, s: str) -> int:
        res = 0
        N = len(s)
        for i in range(1, N):
            l, r = i - 1, i
            leftChar = s[l]
            rightChar = s[r]
            while l >= 0 and r < N and s[l] != s[r] and s[l] == leftChar and s[r] == rightChar:
                res += 1
                l -= 1
                r += 1

        return res
```

- time: O(n^2)
- space: O(1)

## Editorial

### Approach #1: Group By Character [Accepted]

**How it works:**

1. First, we group consecutive identical characters and count their lengths
2. Then, we count valid substrings between adjacent groups
3. The number of valid substrings between two groups equals the minimum of their sizes

**Example:** For `s = "00110011"`

Step 1 - Grouping:

- "00" → group of 2 zeros
- "11" → group of 2 ones
- "00" → group of 2 zeros
- "11" → group of 2 ones
- Result: `groups = [2, 2, 2, 2]`

Step 2 - Counting:

- Between groups[0]=2 and groups[1]=2: min(2,2) = 2 substrings ("01", "0011")
- Between groups[1]=2 and groups[2]=2: min(2,2) = 2 substrings ("10", "1100")
- Between groups[2]=2 and groups[3]=2: min(2,2) = 2 substrings ("01", "0011")
- Total: 2 + 2 + 2 = 6

### Approach #2: Linear Scan [Accepted]

```python
class Solution(object):
    def countBinarySubstrings(self, s):
        ans, prev, cur = 0, 0, 1
        for i in xrange(1, len(s)):
            if s[i-1] != s[i]:
                ans += min(prev, cur)
                prev, cur = cur, 1
            else:
                cur += 1

        return ans + min(prev, cur)
```
