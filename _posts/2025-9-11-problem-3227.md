## Problem Statement

[leetcode problem link](https://leetcode.com/problems/vowels-game-in-a-string/description/?envType=daily-question&envId=2025-09-12)

## Brute Force approach [Accepted]

```python
class Solution:
    def doesAliceWin(self, s: str) -> bool:
        vowels = {'a','e','i','o','u'}
        vowel_count = 0
        non_vowel = 0
        for c in s:
            if c in vowels:
                vowel_count += 1
            else:
                non_vowel += 1

        if vowel_count == 0:
            return False
        if vowel_count % 2 != 0:
            return True

        return True
```

## Editorial

### Approach: Greedy

```python
class Solution:
    def doesAliceWin(self, s: str) -> bool:
        return any(c in "aeiou" for c in s)
```
